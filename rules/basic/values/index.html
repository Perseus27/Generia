    General

    Movement Speed

    Base Value

    MS = 0

    Definition

    Movement Speed is the distance in meters that a creature can move within one movement action.

    Initiative

    Calculation

    INI = PER-MOD + 1d8

    Definition

    Initiative describes the order in which creatures take their turns during combat.

    Energy (EN)

    Base Value

    EN = 0

    Definition

    Energy is a resource describing the physical stamina of a creature. It is usually expended for skills in and out of combat. It can be regenerated by consuming food.

    Mana Points (MP)

    Base Value

    MP = 0

    Definition

    Mana is a mystical resource used by spellcasters to fuel their magic. It regenerates slowly out of combat (see: MP Regeneration).

    MP Regeneration (MPR)

    Base Value

    MPR = 0 each Phase

    Definition

    The amount of MP restored each Phase while out of combat.

    Wound Limit

    Base Value

    Wound Limit (WL) = 3

    Definition

    See Wounds.

    Exhaustion

    Exhaustion is a special type of wound caused by overexertion. Each level of exhaustion reduces the WL by 1. Should the WL reach O (or the total amount of incurred wounds) due to exhaustion, the creature is considered unconscious.

    Recovery

    Four uninterrupted hours of sleep remove one level of exhaustion.

    Wound Treshold

    Base Value

    WT = 1

    Definition

    Exceeding the Wound Treshold with either damage or healing points causes or removes a wound.

    Toughness

    Base Value

    TN = 3

    Definition

    If the damage points caused by status effects like poisoned or bleeding exceed Toughness, the creature takes a wound.

    Armor Value

    Definition

    Armor Value increases the WT against physical damage.

    Calculation

    Armor Value is equal to the armor value offered by equipment, per hit zone.

    Magic Defense

    Definition

    Magic Defense increases the WT against magical damage.

    Calculation

    Magic Defense is equal to the total magic defense value offered by equipment like amulets and rings. It does not vary by hit zone.

    Combat

    Attack Value

    Definition

    The attack value is the base value for any basic attack, melee or ranged.

    Calculation

    [Weapon proficiency] + AGI-MOD * Weapon Hit Rate (HR)

    Block Value

    Definition

    The block value is the base value for block checks.

    Calculation

    [Weapon proficiency] + CON-MOD + STR-MOD

    Block Strength

    Definition

    The block strength is the amount of damage reduction offered by a successful block.

    Calculation

    CON-MOD + STR-MOD

    Dodge Value

    Definition

    The dodge value is the base value for dodge checks.

    Calculation

    AGI-MOD + PER-MOD

    Parry Value

    Definition

    The parry value is the base value for parry checks.

    Calculation

    The parry value is equal to the Attack Value.
